# Wishlist Projekt #

En webbaseret databaseapplikation, der giver brugere muligheden for at oprette og administrere en digital ønskeliste.

## Features ##

### Bruger ###
- Opret konto
- Log ind

### Ønskelister ###
- Opret ønskeliste
- Rediger ønskeliste
- Slet ønskeliste

### Ønsker ###
- Tilføj ønsker med:
  - Titel
  - Beskrivelse
  - Pris
  - Produktlink
- Rediger ønsker
- Slet ønsker

## Teknologier ##

- Java 21
- Spring Boot
- Spring Security
- Thymeleaf
- HTML & CSS
- MySQL

## Installation ##

1. git clone https://github.com/JohnCena1306/WishlistProjekt.git 
2. cd WishlistProjekt
3. mvn spring-boot:run

## Brug ##
Åbn i browser:
http://localhost:8080
